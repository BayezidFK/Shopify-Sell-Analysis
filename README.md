# Shopify Sales Analytics Dashboard

## Project Overview
A comprehensive data visualization and analytics solution for tracking Shopify store performance. This project combines high-level dashboard metrics with granular transaction data for complete sales funnel analysis.

## Data Structure
### 1. Dashboard Summary
- **Aggregated Metrics**: Net sales ($4.18M), average order value ($562.6), customer retention (46%)
- **Visual Analytics**: Geographic distribution, sales trends over time, payment method breakdown
- **Key KPIs**: Lifetime value ($943.6), purchase frequency (1.68), customer segmentation

### 2. Transaction Details 
- **Raw Data Layer**: Individual order records with complete customer and product information
- **Data Fields**: Customer ID, location, product type, payment gateway, transaction amounts
- **Granularity**: Line-item detail supporting drill-down analysis from dashboard metrics

## Purpose
Enable data-driven decision making through integrated visualization of sales performance, customer behavior, and regional trends for Shopify merchants.

## Tech Stack
- Data Visualization: Chart-based dashboard
- Data Storage: Tabular transaction records
- Analytics: Customer funnel metrics & regional performance analysis
